Rubidium is a dynamic, modular and efficient data managing and displaying framework. Rubidium provides a clean, flexible and simple ways of interacting with the many different APIs within the framework. 

Rubidium comes equipped with [DataStore2](https://kampfkarren.github.io/Roblox) for seamless and automatic data saving, without having to worry about coding your databases yourself.

The goal of this framework is to take away the stress of managing lots of data. Rubidium achieves this by storing all your data into fully configurable, managed and accessible stores that can be easily created, configured and maintained.

<h1 align="center">Features</h1>
Rubidium is packed with many useful features here are just some:

- Seamless way of sharing data between any script (even on the client!)
- Viewing and modification of any registered data whilst in-game
- Automated data saving, done with DataStore2

What Rubidium hopes to have:

- Clean and flexible plugin ecosystem
- Intergrated and extensible commandline interface
